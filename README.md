# Laptop Refurb Tycoon

An idle/clicker business simulation game about buying, refurbishing, and selling laptops.

## Stack
- React 18 + Vite
- localStorage save/load
- Capacitor (planned — Android/iOS/desktop)

## Pipeline
Incoming → Audit → Repair → Image → Clean → Pack → Ship

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
